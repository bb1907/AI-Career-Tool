## Description draft

AI Career Tools helps you prepare faster and apply with more confidence.

Use AI-powered tools to generate ATS-friendly resumes, tailored cover letters, structured interview prep, and profile-based career documents from your existing CV.

Key features:

- Build ATS-friendly resume content
- Generate cover letters tailored to a selected role
- Prepare for technical and behavioral interviews
- Import a PDF CV and turn it into a reusable candidate profile
- Save previous outputs and revisit them later
- Unlock premium access for higher usage limits and ongoing generation

AI Career Tools is designed to reduce repetitive work so you can focus on stronger applications and better preparation.
